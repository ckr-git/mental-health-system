# 情绪日记 2.0 - 温馨的心灵港湾

## 🏠 核心理念

**打造一个温馨的"家"，而不是僵硬的"打卡平台"**

这不是一个冷冰冰的记录工具，而是：
- 一个可以随时回家的温暖港湾
- 一个可以和过去的自己对话的时光屋
- 一个可以看见情绪"天气"的心灵观景台

---

## 🎨 全新功能设计

### 1. 心情留言板（替代"反思笔记"）

**理念**：不是反思错误，而是评价、对话、陪伴

#### 功能特性
- **名称**："给{时间}的我留个言"
  - "给昨天的我"
  - "给一周前的我"
  - "给那个难过的我"

- **留言类型**（用户可选）：
  - 💬 随便说说
  - 👍 点个赞
  - 🤗 抱抱你
  - 📝 我想说
  - 🎯 现在的想法

- **互动元素**：
  - 👍 赞同（"对，当时就是这样"）
  - 👎 不认同（"现在想想，没那么严重"）
  - ❤️ 心疼（"当时的我好不容易"）
  - 💪 鼓励（"我比那时更强了"）
  - 😅 释然（"现在想想挺搞笑"）

- **留言墙展示**：
  - 时间轴形式
  - 左边：当时的记录
  - 右边：后来的留言
  - 形成"对话气泡"效果

#### 示例
```
[2025-11-01] 当时的我：
"今天工作搞砸了，感觉自己很失败..."
心情：😔 3/10

[2025-11-03] 现在的我：
"其实没那么糟，老板后来说那是常见问题" 
👍 不认同当时的想法 · ❤️ 心疼

[2025-11-10] 未来的我：
"哈哈，现在这个项目我已经做得很好了"
😅 释然 · 💪 鼓励
```

### 2. 时光信箱（升级版"给未来的信"）

**理念**：多样化的情感表达，不局限于一种形式

#### 信件类型（智能推荐）

根据**近期整体心情趋势**自动推荐：

##### 心情连续向上 + 总体>7分
📮 **表扬信**
```
亲爱的未来的我：
你最近很棒！连续5天心情都在7分以上，
给自己一个大大的赞吧！
记住这种感觉，你值得被肯定。

设定解锁时间：[ 30天后 ]
届时你会收到这封来自过去的鼓励
```

##### 心情波动大 或 总体在4-6分
💌 **期望信**
```
未来的我：
现在的我有点迷茫，但我相信：
30天后的你，一定会...
- [ ] 找到新的爱好
- [ ] 学会一个新技能
- [ ] 认识新朋友

给你一个月，加油！

设定解锁时间：[ 30天后 ]
附上当时的心情记录：[链接]
```

##### 心情持续低落
✉️ **未来的你会感谢现在坚持的我**
```
未来的自己，你好：
如果你在30天后看到这封信，
说明你已经度过了这段艰难时期。

我知道你做到了，因为：
现在的我，虽然很难，但每天都在努力...

相信我，也相信你。

设定解锁时间：[ 30天后 ]
到时候回来看看，你会为自己骄傲
```

##### 心情极好（>8分连续3天）
🎉 **时光胶囊 - 快乐存档**
```
嘿，未来的我！
把这份快乐保存起来，
等你需要的时候再打开。

今天发生了什么好事：
[记录当天的开心事]

这种感觉：[语音/文字记录]

解锁时间：[ 30/60/90天后 or 情绪低谷时自动推送 ]
```

##### 新增：各种创意信件
🎈 **倒计时信** - "距离XX还有N天，现在的我..."
🎯 **目标信** - "给30天后完成XX的我"
🎁 **神秘信** - "不记得写了啥，到时候惊喜"
💭 **梦想信** - "给实现XX梦想的我"

#### 信件状态
- 📬 已寄出（显示倒计时）
- 📭 已送达（可查看）
- 🔒 时光锁（特殊条件解锁）

#### 特殊机制
- **紧急拆信权**：情绪极度低落时（<3分连续3天），系统推送：
  "要不要看看过去的你给你写的信？"
  
- **连锁信件**：回复未来信，形成时间轴对话

### 4. 天气情绪系统（核心创新）

**理念**：让情绪可视化、可感知、可沉浸

#### 天气-情绪映射

根据**心情评分**自动匹配天气主题：

| 心情分数 | 天气主题 | 视觉效果 | 音效 |
|---------|---------|---------|-----|
| 1-2分 | 🌩️ 暴雨雷电 | 黑云密布、闪电、暴雨 | 雷声、暴雨声 |
| 3-4分 | 🌧️ 阴雨绵绵 | 灰色天空、细雨、湿润 | 雨声、水滴声 |
| 5-6分 | ☁️ 多云转阴 | 云层较厚、偶尔阳光 | 风声 |
| 7-8分 | 🌤️ 晴朗有云 | 蓝天白云、温暖阳光 | 鸟鸣、微风 |
| 9-10分 | ☀️ 万里晴空 | 璀璨阳光、蓝天 | 欢快音乐 |

#### 特殊天气状态

**极端情绪触发**：
- **😭 情绪崩溃（<2分）**：
  - 🌑 完全黑暗
  - 压抑的背景音
  - 出现"紧急求助"按钮（联系医生/危机干预）

- **🥰 极度幸福（>9分）**：
  - 🎆 烟花绽放
  - 星空闪烁
  - 彩虹出现

#### 动态粒子效果

根据**记录卡片的状态**显示不同效果：

- **正在经历 🌱**：
  - 雨滴/雪花飘落
  - 云朵飘动
  - 风吹效果

- **已经好转 🌤️**：
  - 云层散开动画
  - 阳光洒落
  - 光晕效果

- **完全度过 🌈**：
  - 彩虹桥出现
  - 星星闪烁
  - 金色粒子

- **我战胜了它 💪**：
  - 烟花爆炸
  - 彩带飞舞
  - 勋章闪光

#### 页面整体天气主题

- **天气跟随心情**：
  - 查看当天记录：显示当天天气
  - 查看历史记录：回放历史天气
  - 整体视图：显示综合天气（近7天平均）

- **天气预报**：
  - 顶部显示"今日心情天气：🌤️ 晴转多云"
  - "本周心情预报：☁️→🌤️→☀️"（基于趋势）

#### 交互动画

- **点击记录卡片**：
  - 天气变化过渡动画
  - 从当前天气 → 记录时的天气
  - 伴随音效切换

- **滚动页面**：
  - 云朵视差效果
  - 雨滴/雪花自然飘落
  - 天空颜色渐变

- **心情输入时**：
  - 实时预览天气变化
  - "如果选择3分，会是这样的天气哦"
  - 引导用户感受不同情绪

```sql
    mood_range_max INT COMMENT '心情范围最大值',
    weather_type VARCHAR(20) COMMENT '天气类型',
    weather_name VARCHAR(50) COMMENT '天气名称',
    
    -- 视觉配置
    bg_color_start VARCHAR(10) COMMENT '背景渐变起始色',
    bg_color_end VARCHAR(10) COMMENT '背景渐变结束色',
    particle_type VARCHAR(30) COMMENT '粒子类型：rain, snow, sunshine, cloud, star',
    particle_config JSON COMMENT '粒子配置参数',
    
    -- 音效配置
    ambient_sound VARCHAR(100) COMMENT '环境音文件',
    
    -- 动画配置
    animation_effects JSON COMMENT '动画效果配置'
) COMMENT='天气主题配置表';
```
