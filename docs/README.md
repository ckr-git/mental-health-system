## 📌 快速导航

### 新手入门
1. 阅读 **../README.md** 了解项目概况
2. 参考 **../QUICKSTART.md** 快速启动项目
3. 查看 **testing/测试账号说明.md** 获取测试账号

1. 查看 **development/** 了解开发历程和计划
2. 参考 **features/** 了解各功能模块的实现细节
3. 遇到问题可查阅 **fixes/** 中的解决方案

1. 查看 **testing/** 目录下的测试文档
2. 使用 **scripts/** 中的测试脚本进行自动化测试
3. 参考各功能文档中的测试章节
